# Paparhoof Color Palette

Reference image: `palette-reference.png`

## Palette

| Token | Name | Hex |
| --- | --- | --- |
| `mane-orange` | Mane Orange | `#E34C18` |
| `horse-black` | Horse Black | `#202423` |
| `saddle-green` | Saddle Green | `#01360A` |
| `barn-beige` | Barn Beige | `#FFF093` |
| `truck-grey` | Truck Grey | `#83A1AC` |
| `sheep-white` | Sheep White | `#FFFDF7` |

## Suggested CSS Variables

```css
--paparhoof-mane-orange: #E34C18;
--paparhoof-horse-black: #202423;
--paparhoof-saddle-green: #01360A;
--paparhoof-barn-beige: #FFF093;
--paparhoof-truck-grey: #83A1AC;
--paparhoof-sheep-white: #FFFDF7;
```

## Notes

- This palette was transcribed from the exported brand palette image.
- Keep `horse-black` and `sheep-white` as the primary text/background pair for strongest contrast.
- `mane-orange`, `saddle-green`, and `barn-beige` work well as accent colors.
