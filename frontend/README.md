# Frontend

This app is now a `Next.js` App Router frontend with local content files.

## Commands

```bash
npm install
npm run dev
npm run build
npm run start
```

## Main folders

- `app/`
  Route files, root layout, and the Next entrypoint.

- `src/components/`
  Shared UI and the client shell.

- `src/views/`
  Reusable route views rendered by the app routes.

- `src/content/`
  All manually editable site content.

- `public/content/`
  Replaceable hero, project, and team images.

## Detailed guide

See `src/content/README.md` for the project object shape and the manual content workflow.
